# giLrv5.github.io

This repository hosts a simple webpage built with HTML5 and CSS animations.
It includes a JavaScript-based fireworks effect rendered on a canvas layer.
Fireworks now launch automatically at random intervals from random positions.
Click anywhere on the page to launch additional bursts of particles.

## Flights Timetable

The `flights.html` page now loads flight timetable data from a local file `out.json` instead of making a network request. The included dataset lists several sample flights for **2025/06/21**. Simply open `flights.html` in a browser to view the timetable. If you replace `out.json` with real data from the airport, update `flights.js` accordingly.
