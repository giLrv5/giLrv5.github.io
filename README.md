# giLrv5.github.io

This repository hosts a simple webpage built with HTML5 and CSS animations.
Click anywhere on the page to trigger a small fireworks effect rendered
with JavaScript. The page also uses responsive layout techniques so it
displays well on mobile and desktop screens.

本倉庫包含一個簡易的 HTML5 網頁，透過 CSS 動畫展現動態背景。
在任意位置點擊即可觀看煙火特效，並採用響應式設計在桌面與行動裝置上皆能正常顯示。
